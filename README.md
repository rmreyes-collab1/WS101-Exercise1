# The 8 Planets Website 🌌

This project is a simple website about the 8 planets in our solar system.

## Pages
- Home (index.html)
- Planets (planets.html)
- Contact (contact.html)

## Features
- Clickable image that goes to the planets page
- Navigation menu
- Styled table of planets
- Contact form with validation

## Note
Make sure all files and images are in the same folder.
